What's This About?

You're going to build real machine learning models on a dataset YOU choose. This will become your first GitHub portfolio project! Right now, focus on finding data that genuinely interests you.

Dataset Requirements

  • 1. Format: CSV file
  • 2. Columns: At least 6+ features
  • 3. Target: A numerical column to predict
  • 4. Size: 1,000 - 100,000 rows
  • 5. Type: Tabular data (no images/audio)
  • 6. Approval: Must be approved by Abishek

Questions to Ask Yourself

  • ? What topics genuinely interest me?
  • ? What number would I want to predict?
  • ? What features might help make that prediction?
  • ? Would I enjoy exploring this data?
  • ? Could I talk about this in an interview?

Avoid Time Series Data

Skip datasets requiring date/time predictions (stock prices, weather, sales trends). We haven't covered time series yet!

Watch for Missing Data

Data doesn't need to be perfect, but if 20-30% or more is missing, it can be frustrating. Check before committing!

Target Must Be a Real Number

Your target needs to be an actual number (price, score) - not categories disguised as numbers (1, 2, 3 for "low, medium, high").

Avoid Overused Datasets

Titanic, Iris, Boston Housing - these are everywhere. Pick something unique that stands out in your portfolio!

Watch for Data Leakage

Avoid features that "give away" the answer - like having "total_cost" when predicting "price".

Minimum Size: 1,000 Rows

Your dataset needs at least 1,000 rows to train reliable models. Ideally aim for 10,000+ rows for better results.

Too Big? Scale It Down

Massive datasets can crash your computer! If it's huge, you can sample or remove rows to make it manageable.

Skip Long Text Fields

Datasets with lots of text columns (descriptions, reviews) aren't ideal. Stick to numerical and categorical data.

Consider Domain Knowledge

Avoid datasets requiring deep expertise you don't have (specialized medical data) - unless it's YOUR field!

Not Just Kaggle!

Found data from work or another website? That's fine! Just meet the requirements and get approval from Abishek.
